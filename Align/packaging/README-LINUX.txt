Brainana Align Desktop for Ubuntu/Linux x64

Launch: run ./brainana-align-desktop from this extracted folder.

Local MRI/CT loading, registration, sessions, and browser-download export are included. Remote workstation access uses the system ssh client and one of: x-terminal-emulator, GNOME Terminal, Konsole, or xterm.

This package was cross-assembled on macOS. Validate it on the target Ubuntu computer before production use.
