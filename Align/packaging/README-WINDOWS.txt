Brainana Align Desktop for Windows x64

Launch: double-click Brainana Align Desktop.exe in this extracted folder. Keep all accompanying files and folders together.

Local MRI/CT loading, registration, sessions, and browser-download export are included. Remote workstation access is intentionally disabled in this initial Windows build pending a validated Windows authentication transport.

This package was cross-assembled on macOS. Validate it on the target Windows computer before production use.
