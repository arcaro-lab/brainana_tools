Refined Instrument — Graphite + Amber

The chosen visual identity for the viewer: warm graphite neutrals under a single amber accent that carries today's signature gold, a humanist face (Source Sans 3) with a monospace (IBM Plex Mono) for every number, and crisp hairline seams. Softer contrast for long sessions. This direction is shipped in viewer/src/style.css; the demo below is kept as the living reference.

What you're looking at

The reference renders the real dashboard layout (toolbar, dual viewport, ROI legend, five-column info read-out, function/morphology controls) with a CSS/Canvas placeholder standing in for the live NiiVue brain render, so the identity can be judged at genuine density.

The identity is already ported: the :root token block lives in viewer/src/style.css (now a full spacing/radius/type/semantic scale), and the canvas colors in viewer/src/ui/visualFieldPlot.ts read those same tokens. See README.md in this folder for the rundown.
