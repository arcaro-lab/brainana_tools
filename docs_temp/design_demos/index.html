What you're looking at

These are throwaway design mockups — none of them touch the app yet. Each page renders the real dashboard layout (toolbar, dual viewport, ROI legend, five-column info read-out, floating Function panel) with a CSS/Canvas placeholder standing in for the live NiiVue brain render, so you can judge each identity at genuine density.

Once you pick one, porting it is a stylesheet-level swap: the :root token block at the top of the chosen file replaces the tokens in viewer/src/style.css, and the hardcoded canvas colors in viewer/src/ui/visualFieldPlot.ts move to match. The class names in these demos already mirror the app's real DOM. See README.md in this folder for the full rundown.
